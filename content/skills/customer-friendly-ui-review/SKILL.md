# Customer Friendly UI Review

## 檢查項目

1. 首屏是否讓使用者知道要做什麼？
2. 手機版是否不重疊、不爆版？
3. 表單錯誤是否有清楚提示？
4. loading、empty、error state 是否完整？
5. 客戶可見文字是否避免過度承諾？
6. 主要操作是否容易找到？

## English Summary

Review customer-facing pages for clarity, responsiveness, error handling, and trustworthy copy.
