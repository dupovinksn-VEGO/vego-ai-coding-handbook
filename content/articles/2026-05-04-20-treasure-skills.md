## 審核建議

1. 先挑 3 個高價值、低風險來源做 read-only repo scan。
2. 優先轉換 `mattpocock-skills`、`superpowers-zh`、`impeccable`，因為它們和 AI Coding 寶典關聯最直接。
3. 內容創作類 Skill 可放入「可選外部工具」，不直接進 coding 核心流程。
4. 所有包含本地 app、npm、shell、發布、下載、監控功能的 Skill 先標中高風險。
