# 20 個寶藏 Skills 外部雷達

## 使用時機

當 VEGO 需要從外部 AI Coding / Skill 社群吸收新工具時，用此 Skill 做第一輪分類、風險標記與 VEGO 適用度判斷。

## 優先審核候選

1. mattpocock-skills：工程化 coding workflow。
2. superpowers-zh：中文 AI 編程流程與 review 規範。
3. impeccable：前端設計反模式，可補強 customer-friendly-ui-review。
4. ljg-skills：中文閱讀、思考、卡片化工具。
5. huashu-skills：內容生產線，適合作為文章與手冊素材工具。

## 安全限制

- 不自動執行 `npx skills add`。
- 不自動安裝 npm package。
- 不自動啟用外部 Skill。
- 先只讀檢查 README、SKILL.md、license、package metadata。
- 需要登入、發布、下載、監控或上傳能力的 Skill 標為中高風險。

## 輸出

- 外部 Skill 評分表。
- VEGO 適用分類。
- 風險分級。
- 是否值得轉換成正式 VEGO Skill。
