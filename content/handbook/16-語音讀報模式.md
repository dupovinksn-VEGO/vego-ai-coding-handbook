# 語音讀報模式

VEGO 寶典需要能在通勤、搭飛機、地鐵、高鐵或長時間工作後使用語音聆聽，避免一直盯著螢幕。第一階段先使用瀏覽器內建 Web Speech API 朗讀章節內容，不需要後端服務，也不會增加雲端成本。

## 使用方式

1. 打開任一寶典章節。
2. 在閱讀器上方選擇朗讀速度。
3. 按「朗讀」開始聆聽。
4. 需要中斷時按「暫停」或「停止」。
5. 切換章節時，系統會自動停止上一章朗讀，避免兩段音訊重疊。

## 適合場景

- 搭飛機時聽寶典重點。
- 搭地鐵或高鐵時聽 AI Coding、物流 AGI、Skill Vault 更新。
- 長時間看螢幕後，用耳朵複習知識庫。
- 週末或睡前聽文章搜集區整理出的摘要。

## 現階段限制

- 語音品質取決於使用者裝置與瀏覽器。
- 某些瀏覽器可能沒有繁中語音，會使用系統可用的中文或英文語音。
- 網頁版朗讀通常需要使用者先點擊按鈕，不能自動播放。
- 超長章節會先朗讀主要內容，未來可拆成段落佇列。

## 後續升級方向

### Edge TTS 離線音檔

參考 VEGO 佛堂輔助工具的做法，重要章節可以用 Microsoft Edge TTS 預先轉成 MP3，放入 `/audio` 類資料夾，形成「線上朗讀 / 離線音檔」雙模式。這適合飛機、網路不穩、高鐵訊號弱或需要更穩定音質的場景。

第一階段仍先使用瀏覽器 Web Speech API，因為寶典章節會快速迭代；等章節穩定後，再挑高價值內容轉成離線音檔。

### AI 摘要讀報

把長章節、外部文章、電子書摘要先壓縮成 3 到 8 分鐘的「通勤版重點」，再朗讀。

### 每週智庫播報

每週自動整理新增文章、Skill、物流自動化案例與 AGI 進展，產生一份「VEGO AI 智庫週報」。

### 離線音檔

未來可把重要章節轉成 MP3，讓搭飛機或網路不穩時也能聽。

### 角色化讀報

依照角色選擇播報內容：

- 老闆模式：策略、風險、投資優先順序。
- 工程模式：架構、測試、部署、Skill。
- 營運模式：物流流程、異常、客服、關務、倉庫。
- 學習模式：名詞、案例、練習題。

## English Summary

Voice briefing mode lets users listen to handbook chapters while traveling or resting their eyes. The first version uses browser speech synthesis, with future upgrades for AI summaries, weekly briefings, offline audio, and role-based listening.
