# Prompt 學習指南

## VEGO Prompt 公式

```text
Goal + Context + Constraints + Done when
```

## 每次寫 Prompt 都要檢查

- Goal 是否明確？
- Context 是否包含業務規則、資料欄位、錯誤訊息？
- Constraints 是否說清楚不可改 production、不可改無關檔案？
- Done when 是否包含測試、build、README、API response 或頁面驗收？

## English Summary

Reliable AI coding prompts must include goal, context, constraints, and concrete completion criteria.
