# 智庫來源庫

這裡存放 VEGO 想要吸收的外部來源與自有系統來源。

## 可以放什麼

- 外部電子書或訂閱內容的連結與摘要。
- 使用者自己創建的系統 URL。
- GitHub repo。
- 官方 docs。
- 外部文章與工具頁。

## 邊界

- 不繞過 DRM、paywall 或授權限制。
- 不整本複製外部電子書。
- 不自動執行外部命令。
- 對自有系統先做 read-only 吸收。

## English Summary

Source library for turning external materials and VEGO-owned systems into reviewed handbook knowledge and reusable skills.
