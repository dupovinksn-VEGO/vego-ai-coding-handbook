# VEGO Logistics AGI Domain Architecture

## 吸收目標

建立 VEGO 智庫的長期領域分層，讓文章、電子書、系統 URL、GitHub Skill 與營運案例都能被放到正確的位置，逐步支援 AI 自動化、智能體、機器人協作與物流 AGI 控制塔。

## 領域分層

- Knowledge：文章、SOP、案例、電子書摘要。
- Automation：文件抽取、狀態比對、異常分類、報表與草稿。
- Agents：Inbound、Warehouse、Customs、Dispatch、Customer、Finance。
- Robotics：掃碼、盤點、AMR、視覺檢查、現場安全停止。
- Control Tower：跨流程監控、模擬、調度建議、升級規則。
- Safety：Sandbox Review、人工批准、合規邊界、責任追溯。

## 下一步

1. 每個來源進入智庫時，都標記所屬層級與營運領域。
2. 每個 Skill 都補上使用限制、輸入、輸出、測試案例與是否可正式啟用。
3. 每週更新時優先收集物流 AI、自動化、機器人、供應鏈 Agent 與關務智能化內容。
