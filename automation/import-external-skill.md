# 外部 Skill 匯入 SOP

## 輸入類型

- 文章全文
- URL
- GitHub repo
- `npx skills add ...` 安裝命令
- 已下載的 Skill 資料夾

## 匯入步驟

1. 抽取 repo、作者、Skill 名稱、使用場景。
2. 下載 README、SKILL.md、license、package metadata。
3. 不執行任何外部腳本。
4. 產生 Sandbox Card。
5. 產生 VEGO 轉換建議。
6. 等待人工批准。

## English Summary

External skill import is read-only first and produces sandbox review cards before approval.
